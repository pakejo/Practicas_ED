Descripción detallada

template<class T, class U>
class Diccionario< T, U >

T.D.A Diccioanrio.

Una instancia c del tipo de datos abstracto Diccioanrio es un objeto con un campo, una lista de data , formada por una clave y su informacion asociada La lista contendrá una serie de data ordenadas por la clave de cada una
